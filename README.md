## Usage
1. Add the project to Unity.
2. Open the project and go to Assets -> Scenes -> Project
3. In the Level Generator Component, choose Number Of Colors and Empty Bottles (The Total is the number of bottles that will be in the game). Choose the Shuffle Steps (The higher the more difficult)
